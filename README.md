# RGC DayZ Configurator

Утилита для управления DayZ сервером — от установки до мониторинга.  
Современный интерфейс, система проектов, встроенный SteamCMD, RCon, генератор конфигов и многое другое.

---

## 📦 Возможности

### 🖥 Полный конфигуратор сервера
Десять вкладок настроек: основные параметры, мир, сеть, моды, batch, конфиги модов, запуск, RCon, статистика.  
Все поля соответствуют параметрам `serverDZ.cfg` и `StartBat.cmd` — сохраняются прямо в папку сервера.

### 📁 Система проектов
Создавайте отдельные проекты для разных серверов.  
Каждый проект — своё название, своя папка, свои моды.  
Переключение между проектами — один клик.

### ⚡ Установка сервера через SteamCMD
Выбрали папку — нажали кнопку. SteamCMD сам скачает, установит и обновит DayZ Server.  
Без консоли, без ручного ввода команд.  
Поддержка Steam Guard (двухфакторной аутентификации).

### 🎨 Красивый интерфейс
- Тёмная и светлая темы
- Плитки проектов с анимацией появления
- Плавные ховеры и переходы
- Иконки и визуальная обратная связь

### 🔐 Генератор
Не можете придумать название сервера? Нажмите 🎲 — получите готовое:  
`Last Horizon`, `Зона Отчуждения`, `Стальной Легион`, `Whisper Woods` и ещё 80+ вариантов.  
Пароль тоже сгенерирует — криптостойкий, 16 символов, со спецсимволами.  
Описание сервера — шаблонные тексты для PvP, RP, Hardcore или Vanilla+.

### 🌍 Локализация
Русский / English — переключается в настройках.  
Тексты основных элементов интерфейса, уведомления и подписи обновляются сразу.

### 🔍 Проверка версии DayZ
Кнопка «Версия» на вкладке Запуск:  
сравнивает установленную версию `DayZServer_x64.exe` с актуальной в Steam.  
Показывает уведомление — всё актуально или пора обновить.

### 📤 Экспорт / Импорт конфига
Сохраните все настройки сервера в `.rgc` файл.  
Перешлите другу — он импортирует конфигурацию одной кнопкой.  
Подхватывается папка сервера, моды и профили.

### 🛠 RCon
Подключение к серверу по протоколу Source RCON.  
Отправка команд, просмотр ответа в реальном времени.  
Пароль сохраняется в конфиге.

### 📊 Статистика игроков
Парсинг `.adm` файлов из папки `profiles`.  
Время игры каждого игрока, статус (онлайн/офлайн), последний визит.

### 🧹 Валидация путей
Перед запуском сервера проверяет:  
- `DayZServer_x64.exe` существует  
- `serverDZ.cfg` существует  
- Папки `battleye` и `profiles` на месте  

Если чего-то нет — всплывают уведомления, сервер не запустится.

## 🚀 Быстрый старт

### 1. Установка
Скачайте последний релиз со страницы [Releases](https://github.com/LokiRGC/RGC-Configurator/releases).  
Распакуйте, запустите `RGC.exe`. .NET встроен — ничего дополнительно устанавливать не нужно.

### 2. Создайте проект
Нажмите «＋ Новый проект», введите название (или сгенерируйте 🎲).  
Проект создастся, вы сразу попадёте в главное окно.

### 3. Выберите или установите сервер
- **Установить новый:** вкладка «Запуск» → «Установить сервер» → выберите папку.  
  SteamCMD скачает сервер автоматически. Потребуется авторизация в Steam.
- **Подключить существующий:** нажмите «Обзор» и укажите папку с уже установленным DayZ Server.

### 4. Настройте конфиг
Заполните поля в 10 вкладках → нажмите «Сохранить» внизу.  
Файлы `serverDZ.cfg` и `StartBat.cmd` запишутся в папку сервера.

### 5. Запустите сервер
Вкладка «Запуск» → «▶ Запустить».  
Лог сервера отображается в реальном времени.
